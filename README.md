Telegram Google Drive Bot
Это телеграм-бот, который позволяет управлять файлами в Google Drive: копировать, перемещать, удалять и изменять права доступа.
Бот использует библиотеку pyTelegramBotAPI (telebot) для работы с Telegram и Google Drive API для работы с файлами.

Основные возможности
/start — приветственное сообщение и кнопки для основных команд.
/copy — копирует все файлы из одной папки в другую.
/move — перемещает файл в указанную папку.
/delete — удаляет файл (с подтверждением).
/setpermissions — изменяет права доступа к файлу (reader, writer, owner).
/cancel — отменяет текущую операцию на любом шаге.
Структура проекта
config.py
Хранит конфигурационные данные, например токен бота и Google ID (если нужно).
drive_service.py
Содержит функции для работы с Google Drive API: авторизация, извлечение ID из ссылки, копирование, перемещение, удаление, изменение прав доступа.
main.py
Основной код бота на pyTelegramBotAPI. Здесь описаны команды и логика FSM (состояний) для пошагового взаимодействия с пользователем.
requirements.txt
Список зависимостей, необходимых для запуска бота.
